JSON-RPC Commands

This RPC API document is compatible with ckb 0.14.0 (v0.14.0 2019-06-15). For more versions of this document, please refer to the GitHub repo directly.

Here is a paw file you can use to import into Rest API softwares such as Postman

The RPC port of your CKB node can be set in your node configuration file. By default, it uses port 8114.

An Example

You can interact with your CKB node client via RPC port. Here's an example showing how to get the header information of the tip block (the latest block).

Open a new terminal, and use this command to get the tip block header (note that you should have the ckb run running):

curl -d '{"id": 2, "jsonrpc": "2.0", "method":"get_tip_header","params": []}' -H 'content-type:application/json' 'http://localhost:8114'

Note that you need to have curl installed.
